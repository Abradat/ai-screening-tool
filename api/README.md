# AI Screening Tool API

## Usage

```
fastapi dev src/api/main.py
```

To view/try the API:

```
http://127.0.0.1:8000/docs
```

For a little nicer API docs and to download OpenAPI spec:

```
http://127.0.0.1:8000/redoc
```

## Installation

```
python3 -m venv venv
source venv/bin/activate
pip install .
```

## Development

```
python3 -m venv venv
source venv/bin/activate
pip install -e .
```
